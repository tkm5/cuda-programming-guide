## 概要

この記事では，コースで使用するGitHubリポジトリについて紹介する．リポジトリにはすべてのプロジェクトのソースコードが含まれており，各セクションで学ぶ内容に対応したサンプルコードを入手できる．

## 主要な内容

### リポジトリの概要

コースのGitHubリポジトリには，各セクションのプロジェクトに対応するCUDAソースコードが格納されている．リポジトリをクローンすることで，ローカル環境でコードを実行し，動作を確認しながら学習を進めることができる．

### リポジトリの使い方

リポジトリの取得は以下の手順で行う．

```bash
git clone <リポジトリURL>
```

クローン後，各プロジェクトのディレクトリに移動して，対応するCUDAソースファイル（`.cu`拡張子）を確認する．ソースコードをエディタで開き，レクチャーの内容と照らし合わせながら学習を進めることが推奨される．

### ディレクトリ構成

リポジトリはセクションごとにディレクトリが分かれており，各プロジェクトのソースコードが整理されている．レクチャーで解説するコードと同じ内容がリポジトリに含まれているため，手入力の手間を省きつつコードの理解に集中できる．

## まとめ

- コースのGitHubリポジトリにはすべてのプロジェクトのCUDAソースコードが含まれている
- `git clone`でリポジトリを取得し，ローカル環境で実行できる
- 各セクション・プロジェクトごとにディレクトリが整理されている
